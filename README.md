# 使い方

1. terminalを開き、以下のコマンドを実行する。
```
$cd frontend/
$yarn start
```
2. 別のterminalを開き、以下のコマンドを実行する。
 ```
$cd backend/
$python manage.py runserver
```
3. ブラウザでhttp://localhost:3000/にアクセスする。
